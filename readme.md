# Run application

**note:** to run the application you need custom dev server(won't work on expo go)

Clone the repository 

```bash
git clone https://github.com/Rohit-00/chat-assignment.git
```
Install dependencies
```bash
npm install
```
Run the application
```bash
npx expo run android
```
on IOS
```bash
npx expo run ios
```
